# Kasm Custom Workspaces

Repo này chứa nhiều custom image cho **Kasm Workspaces**, dựa trên các base image chính thức của Kasm.

### Các image hiện có
- **debian-full** → Debian Bookworm + fcitx5-unikey (tiếng Việt), Zsh + Oh My Zsh (Powerlevel10k + nhiều plugin), VS Code, WezTerm, devtools.
- **ubuntu-full** → (working)

### Pull image
```bash
docker pull ghcr.io/anvu69/kasm-custom-workspace/debian:latest
